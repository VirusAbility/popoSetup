## 마인크래프트 포포 놀이터 Setup 가이드

아래 규칙대로 진행하지 않아 발생하는 문제는 본인 책임임을 명시함.

***

#### 1. 설치
1-1. 런처 설치

https://github.com/VirusAbility/popoSetup/releases/download/1.0.0/Pixel.Launcher.exe

1-2. 리소스팩 설치

http://ineze.kr/popo/pixel.zip
